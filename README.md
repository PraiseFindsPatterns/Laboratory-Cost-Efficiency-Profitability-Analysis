# Laboratory-Cost-Efficiency-Profitability-Analysis
How can we reduce experiment costs while maximizing profitability and efficiency?

## Business Problem
Laboratory experiments incur various costs (materials, energy, and lab usage), but not all experiments generate equal value.

This project analyzes how costs impact profitability and identifies which experiments are efficient, wasteful, or break-even.

---

## Objectives
- Evaluate total cost vs revenue for each experiment  
- Identify profit-generating vs loss-making experiments  
- Measure efficiency using key performance indicators  
- Provide recommendations to optimize resource allocation  

---

## Dataset
The dataset contains 7 laboratory experiments with features such as:
- Quantity of materials used  
- Cost per unit  
- Energy and lab costs  
- Revenue generated  

---

## Key Metrics
- **Total Cost (TC)** = Material + Energy + Lab Cost  
- **Profit (P)** = Revenue − Total Cost  
- **Efficiency Index (EI)** = Revenue / Total Cost  

---

## Key Insights
- Some experiments generate high revenue but are inefficient due to high costs  
- A few experiments operate at a loss, indicating poor resource allocation  
- High efficiency does not always mean highest profit  
- Optimizing material usage significantly improves profitability  

---

## Visualizations
The project includes:
- Profit comparison across experiments  
- Cost breakdown analysis  
- Efficiency index ranking  

---

## Tools Used
- Microsoft Excel  
- Python (Pandas, Matplotlib) *(optional upgrade)*  

---

## Recommendations
- Reduce material waste in high-cost experiments  
- Focus on high-efficiency experiments  
- Re-evaluate experiments with consistent deficits
